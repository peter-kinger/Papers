# social/climate model/system
> social climate model 和 world earth model 都是在全球尺度进行研究的
>
> “ESM的发现缺少了一个关键的组成部分。地质学家目前正考虑将当前时期称为“人类世”，因为人类对地球的影响是前所未有的，并且还在不断加剧环境的影响。为了模型的准确性，必须考虑人为因素。因此，人类世界有时会被整合到ESM中，形成一类称为“世界-地球系统模型”的不同类别的模型。这类模型最早的实例来自1974年的《增长的极限》[46].这本书通过一个动态模型——World3，描述了人类可能的不同路径，该模型考虑了人口增长、经济和污染。“世界-地球系统模型”是波茨坦气候影响研究所（PIK）使用的术语。该研究所进行了大量关于世界-地球系统建模的研究，例如我们在本工作中使用的AYS模型[35]。另一种术语是社会-气候模型[31,32]。这项研究仍然相当少见，命名尚未完全确定。为了清晰起见，我们将称其为世界-地球系统”
